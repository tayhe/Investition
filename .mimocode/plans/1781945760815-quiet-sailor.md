# 修复期权计算逻辑

## 问题
期权的 avgCost、costBasis、marketValue、pnl 计算混乱，单位不统一。

## 用户定义的正确逻辑

| 字段 | 单位 | 存储方式 | 计算方式 |
|------|------|---------|---------|
| avgCost | **每股** | 存入 DB | IBKR 原始数据（不做任何转换） |
| currentPrice | **每股** | 不存 DB | Yahoo 原始数据（不做任何转换） |
| costBasis | **实际成本** | 不存 DB | \|quantity\| × multiplier × avgCost |
| marketValue | **实际市值** | 不存 DB | \|quantity\| × multiplier × currentPrice |
| pnl | **实际盈亏** | 不存 DB | marketValue - costBasis（做多）/ costBasis - marketValue（做空） |

其中 multiplier: 股票=1, 期权=100

## 当前代码问题

### 1. FIFO (`src/lib/ibkr/fifo.ts`)
- **当前**: `pricePerContract = pricePerShare * multiplier`，avgCost 存储为每手价格（$1045）
- **应改为**: 直接用每股价格计算，avgCost 存储为每股价格（$10.45）

### 2. sync upsertPositions (`src/lib/ibkr/sync.ts:260`)
- **当前**: `costBasis = averageCost * |quantity|`（没乘 multiplier，所以 costBasis 是每股×手数=错的）
- **应改为**: 不存 costBasis 到 DB（改为页面计算），或存 `averageCost * multiplier * |quantity|`

### 3. Dashboard (`src/app/(app)/page.tsx:30-53`)
- **当前**: `avgCost = costBasis / absQty / multiplier`（从 DB 的 costBasis 反推每股价格）
- **应改为**: 直接用 `pos.avgCost`（已经是每股价格）

### 4. Portfolio (`src/app/(app)/portfolio/page.tsx`)
- 同样的问题

## 修复方案

### Step 1: FIFO 存每股价格
文件: `src/lib/ibkr/fifo.ts`

```ts
// 删除: const pricePerContract = pricePerShare * multiplier;
// 改为: 直接用 pricePerShare
const totalCost = qty * pricePerShare + commission;
// avgCost = totalCost / totalQty（每股价格）
// costBasis = avgCost * absQty * multiplier（实际成本，用于写 DB）
```

### Step 2: sync 不再写 costBasis
文件: `src/lib/ibkr/sync.ts` (upsertPositions)

```ts
// avgCost 保持每股价格（来自 Flex 或 FIFO）
// costBasis = 0 或不写（由页面实时计算）
```

### Step 3: Dashboard/Portfolio 统一计算
文件: `src/app/(app)/page.tsx`, `src/app/(app)/portfolio/page.tsx`

```ts
const avgCost = Number(pos.avgCost);        // 每股，直接用
const currentPrice = priceMap.get(...)       // 每股，Yahoo 原始
const multiplier = type === "OPTION" ? 100 : 1;
const costBasis = absQty * multiplier * avgCost;      // 实际成本
const marketValue = absQty * multiplier * currentPrice; // 实际市值
const pnl = isShort ? (costBasis - marketValue) : (marketValue - costBasis);
```

### Step 4: Snapshot 计算也统一
文件: `src/lib/ibkr/sync.ts` (createDailySnapshot)

当前 `positionsValue += quantity * multiplier * price` 已正确。

## 验证

1. `npm run build` 无错误
2. 重新处理缓存后检查期权数据：
   - avgCost 应为每股价格（如 $10.45）
   - costBasis 应为 |qty| × 100 × avgCost
   - marketValue 应为 |qty| × 100 × currentPrice
3. 做多股票：pnl = marketValue - costBasis
4. 做空期权：pnl = costBasis - marketValue（价格下跌=盈利）
